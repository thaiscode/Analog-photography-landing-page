💼 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de simular um cenário real de entrega de um site institucional/portfólio, priorizando:

Clareza na apresentação de serviços
Hierarquia visual bem definida
Layout responsivo
Código organizado e reutilizável

A landing page apresenta informações sobre a fotógrafa, seus diferenciais e uma galeria de trabalhos, além de uma seção de contato.

🚀 Principais funcionalidades
 - Navegação simples e intuitiva
 - Seção hero com CTA
 - Apresentação de marcas/equipamentos
 - Seção de diferenciais
 - Galeria de portfólio
 - Layout responsivo
 ⚠️ Estrutura de formulário está pronta, mas sem integração com backend.

🛠️ Tecnologias 
HTML5 semântico
CSS3 
Flexbox
Responsividade
